### instrucyion to start chat server

install compiler  MinGW (https://www.mingw-w64.org/).
confirm that the  gcc in command line (команда: gcc --version).
clone and download/ folder TCP_Project_Chat.
run build_chat.bat (double click).
 run `server_chat.exe`.
 run `client_chat.exe` (you can open a couple windows chats).
use the chat enter the messages
 all messages is saved in to the `server_log.txt`.

look that the port are not blocked antivirus or browser programs
